Rikka - The Java Bot.

Info about Rikka.

Rikka is a discord bot created for the sole reason of fun. I created her as just a random project to pass the time but she ended up being a whole lot more useful when I finished. She has over 50 commands, with room for more. Soon, I'll be adding a chat filter option.

Info about the developers

The developers include Ethan Buckton (Lucifer#6159) and Zoey Globe (ZoeyGlobe#6849). We're both college students, who thoroughly enjoy programming. We're both versed in many languages. We both run Linux desktops; Ethan runs Linux Ubuntu MATE 18.04, and Zoey runs Linux Xubuntu 18.04.

Other info.

Rikka is made with the JDA libary, and uses several APIs. These APIs include; the imageboard-api by Kodehawa and lavaplayer by Sedmelluq.
